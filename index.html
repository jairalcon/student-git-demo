Student Git Demo

    • Create a remote repository on GitHub.
      Create a local repository on your computer git init
      Change the name of the default branch to match Github git branch -M main
      Connect the repository on your computer to the remote repository git remote add origin https://your-repo-name.git
      Make some changes to the files in that folder
      Commit every time you get a piece working git add . git commit -m “Added something”
      Push your changes to the online repository git push -u origin main
    • You can check the status of your source control at any time: git status The default branch is generally called main not master now. Private repositories are free on Github now.
