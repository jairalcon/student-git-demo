Student Git Demo

  1. Create a remote repository on GitHub.
  2. Create a local repository on your computer git init
  3. Change the name of the default branch to match Github git branch -M main
  4. Connect the repository on your computer to the remote repository git remote add origin https://your-repo-name.git
  5. Make some changes to the files in that folder
  6. Commit every time you get a piece working git add . git commit -m “Added something”
  7. Push your changes to the online repository git push -u origin main
You can check the status of your source control at any time: git status The default branch is generally called main not master now. Private repositories are free on Github now.
